Pertamina Offshore Drilling Engineering Suite — Demo interaktif aplikasi Drilling Engineering berbasis ReactJS, TypeScript, dan Python (FastAPI).

Empat modul utama dalam satu workspace:

 Drilling Calculator — kalkulasi hidrolika, volumetrik, torque & drag, dan well control
 
 Petroleum Unit Conversion — konversi tekanan, densitas, suhu, dan volume
 
 Well Control Kill Sheet — perhitungan IADC-style (KMW, ICP, FCP, MAASP)
 
 Wellbore Geometry — kapasitas annulus & skema sumur
